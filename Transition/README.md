# Transition/  — the DROP ZONE

Drop anything unsorted here: a file, a screenshot, a link, a half-formed thought. Then ask
your AI to sort it. It files each item into the right home (`context/`, `references/`,
`decisions/`, `projects/`, `templates/`, `archives/`) and **leaves this folder empty**.

An empty `Transition/` means "nothing pending." That's the goal.
